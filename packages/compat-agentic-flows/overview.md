# agentic-flows

`agentic-flows` is a compatibility package that installs `bijux-canon-runtime`.
It also preserves the legacy `agentic_flows` Python import surface.

Use the canonical package for new installs:

```bash
pip install bijux-canon-runtime
```

The legacy command name `agentic-flows` remains available as an alias after installation.
The legacy Python import `import agentic_flows` also remains available through this package.
