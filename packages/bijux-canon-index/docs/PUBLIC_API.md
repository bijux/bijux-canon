# PUBLIC_API

Package-facing surfaces in `bijux-canon-index` are:
- Python package: `bijux_canon_index`
- API builder: `bijux_canon_index.api.v1.build_app`
- API schema artifacts under `apis/bijux-canon-index/v1/`
- plugin entrypoint contracts documented for vector stores, embeddings, and runners

Internal orchestration helpers under `application/orchestration/` are implementation detail unless explicitly documented otherwise.
