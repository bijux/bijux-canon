# Public API

## Supported entrypoints

- console scripts: `bijux-canon-reason`, `bijux-rar`
- Python package: `bijux_canon_reason`
- CLI module: `bijux_canon_reason.interfaces.cli`
- API package: `bijux_canon_reason.api.v1`

## Not automatically public

Internal modules under `planning/`, `execution/`, and `verification/` are
implementation detail unless the package explicitly documents them as stable.
