# Code of Conduct

We are committed to a welcoming, safe, and respectful community across the
entire `bijux-canon` repository.

## Scope

This code of conduct applies to:

- all repository discussions, issues, pull requests, and reviews
- package-specific documentation and examples
- public spaces where someone is representing this repository

## Expected Behavior

- Be respectful, direct, and constructive.
- Assume good intent while still addressing real problems clearly.
- Help keep collaboration accessible for people with different levels of
  context and experience.
- Accept feedback in the spirit of improving the repository and the community.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination
- Personal attacks, insults, or demeaning language
- Sharing private information without permission
- Deliberately disruptive, misleading, or bad-faith participation

## Reporting

To report a code of conduct issue, contact
[`bijan@bijux.io`](mailto:bijan@bijux.io).

All reports will be reviewed as promptly and fairly as possible.
