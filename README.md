# bijux-llm-nexus

Unified repository for Bijux LLM packages and applications.

Imported projects live in stable top-level directories so each codebase
keeps a clear identity while sharing one repository history.

Projects:

- `agentic-flows`
- `bijux-agent`
- `bijux-rag`
- `bijux-rar`
- `bijux-vex`

History preservation details are documented in `docs/repository-history.md`.
